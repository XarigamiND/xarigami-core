Description

The Dropdown Select property provides a drop down selection list with options that are user definable.

Attributes

  • class: [default='']
    The class can be specified and will be applied to the select tag that is generated from the dropdown data property.
  • firstline: [default='']
    This attribute enables you to specify a first line in the selection list displayed as an instruction to the user.
  • function: [default='']
    Add a xarigami function that delivers an array of values to populate the drop down selectiont list eg 
     xarModAPIFunc('articles','user','dropdownlist',array('ptid'=>1));
  • options: [default=array()]
    This list can be specified as a non-associative or associative array of values to populate the drop down selection list. eg 
     1,red;2,blue;3,green;4,orange
    or 
     red;blue;green;orange

Other attributes may be inherited from the parent property.

Usage

Standalone

You can use the dropdown type to capture data in forms without the need to define a property. 

        <xar:data-input type="dropdown" name="myfield" id="myfield" options="$myoptions"
            firstline="Please select" [attr1="value1",...] />

The 'options' array is in the form of a non-associative or associative array of id=>'value' options eg:

array(1=>'red', 2=>'blue', 3=>'green')

Note that 'validation' is an equivalent legacy attribute and still honoured at this time.

Defined Data Property

You can capture data for this bound property in your templates with the data input tag. 

        <xar:data-input property="$properties['dropdownselect']"
            [attr1="value1",...] />

Example

        <xar:data-input type="dropdown" id="myitems"
            name="myitems" firstline="Please select item" />
