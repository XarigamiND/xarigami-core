Description

The Textarea_medium property provides a textarea input box for text entry. The textarea input box is 8 (eight) rows in height by default.

The Textarea_display output function displays the content of the textarea prepped for HTML output.

The actual value of the text is stored in the database.

Attributes

  • default: [default = '']
    This is the default text displayed in the textarea on input.
  • rows: [default = '8']
    This is the number of rows in height of the textarea.
  • cols: [default = '35']
    This is the number of columns in the text area. It is equated to a style in the textarea property template for better compliance with xhtml standards.
  • class: [default = 'xar-form-text*']
    If defined, the class will be added to the text input tag and applied otherwise a default class is used determined by the number of rows in the text box. The fallback default can be overridden by adding an empty space character to the class input box.
  • max_length: [default = '']
    If defined, this will restrict input to a maximum number of characters specified by maxlength, and display a 'count down' character counter in the input template.
  • min_length: [default = '']
    If defined, this will require input of a minimum number of characters specified by minlength.

Textbox_medium is an 'alias' of textbox property.

Usage

Standalone

You can use the textbox type in forms without the need to define a property to capture data. As well as the attributes available in GUI configurations above, standalone properties also accept valid HTML5 and ARIA WAI attributes passed in the tag. 

        <xar:data-input type="textarea_medium" default="$defaulttext" name="myfield"
            [attr1="value1",...] />

Defined Data Property

When you define this property you can add a default text value in the 'Default' input box. You can also edit the Configuration and enter the rows, cols, and class in the relevant input boxes. The maxlength is entered in the 'Other' input box as maxlength=X.

In your template forms you can use the data-input tag to capture data for this property. 

        <xar:data-input property="$properties['textarea_mediumname']"
            [attr1="value1",...] />

Example

        <xar:data-input type="textarea_medium" rows="4" id="myfield"
            name="myfield" default="Any text" />
