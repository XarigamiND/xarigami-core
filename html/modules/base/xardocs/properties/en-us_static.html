Description

The Static property does not provide any input options when used in a form. You may enter a default value which is displayed on input and output display.

The output display function shows the text as input in the default value.

The value stored in the default value configuration field is stored in the database.

Attributes

  • default: [default = '']
    This is the value of the static text displayed in both input and output display.

Other attributes may be inherited from the parent property.

Usage

Standalone

You can use the static type in forms without the need to define a property to capture data. 

        <xar:data-input type="static" default="$defaulttext" name="myfield"
            [attr1="value1",...] />

Defined Data Property

When you define this property you can add the static text in the 'Default' input box.

You cannot capture data in forms. The Static property will only output the static text display. 

        <xar:data-input property="$properties['static']"
            [attr1="value1",...] />

Example

        <xar:data-input type="static" id="mystatic"
            name="mystatic" default="Any text" />
