Description

The Countrylisting property provides a drop down selection list of countries.

The Country names are displayed in the dropdown list while the ISO-3166 country code is the value captured in forms. On output the selected country name is displayed.

Attributes

  • firstline: [default='Please select']
    This attribute is inherited from the parent 'dropdown' selection property, and enables you to specify a first line in the selection list displayed as an instruction to the user.

Other attributes may be inherited from the parent Dropdown List property.

Usage

Standalone

You can use the countrylisting type to capture data in forms without the need to define a property. 

        <xar:data-input type="countrylisting" name="myfield"
            [attr1="value1",...] />

Defined Data Property

You can capture data for this bound property in your templates with the data input tag. 

        <xar:data-input property="$properties['countrylisting']"
            [attr1="value1",...] />

Example

        <xar:data-input type="countrylisting" id="mycountry"
            name="mycountry" firstline="Please select your country" />
