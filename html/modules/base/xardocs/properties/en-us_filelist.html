Description

The Filelist property provides a drop down selection input display of a list of files from a given base directory. The file types can be specified and whether or not to display the file extensions in the drop down selection box.

The value of the Filelist property is the file name which is also output on display.

Attributes

  • basedir: [default='var']
    The directory where the files to be displayed in the drop down selector are located.
  • extensions: [default='']
    Allowed file extension types of files for display in the drop down. Separated by commas.
  • longname: [default='false']
    Show the file extension of files in the drop down selector.

Other attributes may be inherited from the Dynamic_Select_Property (dropdown) parent property.

Usage

Standalone

  • In your template forms: you can use the data-input tag to capture data for this property without the need to define the property first elsewhere. 
        <xar:data-input type="filelist" file_ext="pdf,txt" basedir="var/uploads" name="myfield"
            [attr1="value1",...] />

Defined Data Property

  1. Defining your data property: you can define the options for File extensions, Base directory and Display of file extension, in the Configuration screen for this property.
  2. Using the data-input tag: you can use the data-input tag to capture data for this property and pass in attributes for the data property that will override those defined in the Configuration. 
        <xar:data-input property="$properties['filelist']"
            basedir="var/uploads" [attr1="value1",...] />

Example

        <xar:data-input type="filelist" id="yourfile"
            name="yourfile" basedir="var/uploads"  file_ext="pdf,txt,doc" longname="true" />
