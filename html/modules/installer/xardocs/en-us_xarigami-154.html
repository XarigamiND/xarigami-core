1.5.4 Notes

Version 1.5.4 includes bug fixes and improvements with some major changes related to themes.

  1. Jquery has been updated to 1.9.1 release
  2. All plugins that have version updates for JQuery core 1.9.x compatibility have been updated including UI, Superfish and treeTable plugins used in core. Please check any override templates, css and javascript that is related to these changes in your themes.
  3. xgami-001069 Bug Dynamic data menulinks and display privileges
  4. It is very important that you have a database BACKUP prior to running this script and check the upgrade documentation. You should also run the Site Health Checks after the intial upgrade step, and before finalizing this upgrade process.
