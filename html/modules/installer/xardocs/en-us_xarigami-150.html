1.5.0 Notes

Version 1.5.0 is a major feature release focusing on security and performance, as well as an assortment of bug fixes.

  1. Removal of the theme_states and module_states table, along with related code adjustments, with status tracked in the modules and themes tables themselves. This results in cleaner code and a performance boost as well.
  2. Further securing of index.php with consolidation and moving of code out to core classes.
  3. Core caching improvements.
  4. Performance adjustments in code for module, theme, user and duv variables.
  5. Role and privilege optimizations and performance adjustments
  6. DB class performance adjustments.
  7. Bootstrap integrity checks and fail safe error page
  8. Please ensure you have a database BACKUP, check the upgrade documentation, and run the Site Health Checks before finalizing this upgrade process.
