1.5.2 Notes

Version 1.5.2 is a bug fix release with some usability improvements.

  1. Improvements in install process in usability and error messages
  2. xgami-001053: Bug - _itemid not passed in data-output tags especially relevant for showview and showlist output for some properties requiring unique itemid for display
  3. xgami-001055: Bug- Deprecated mysql functions replaced
  4. xgami-001057: Bug- mysqli table operation broken
  5. xgami-001051: Improvement- isParent calls method improved performance
  6. xgami-001056: Improvement- Update to adodb 5.1.8a library
  7. xgami-001059: Improvement- DEMO mode should not show table functions
  8. xgami-001052: New feature- Allow admin role edit from Memberlist
  9. It is very important that you have a database BACKUP prior to running this script and check the upgrade documentation. You should also run the Site Health Checks after the intial upgrade step, and before finalizing this upgrade process.
