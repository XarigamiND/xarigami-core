1.3.4 Notes

Version 1.3.4 introduces better support for database creation options, new theme module features and further directory change optimizations.

  1. This upgrade will check your config.system.php file for an Db.Charset entry and try and write it. Please check that your config.system.php file is writeable by this script, or add th DB.Charset line yourself (check config.system.php.dist).
  2. There is some directory changes that should be transparent to use, and a new 'xarigami' directory in the 'lib' directory.
  3. There have been changes to the Administration interface, notably moving of restriction and sitelock to Base module and performance options such as template and CSS caching.
  4. This upgrade process will also check for corrected directories.
