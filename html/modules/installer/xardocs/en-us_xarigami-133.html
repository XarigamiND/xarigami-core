1.3.3 Notes

Version 1.3.3 introduces the themevars scenario to Xarigami allowing a number of useful features for theme creators and site admins.

  1. Theme variables allow creation of more portable themes, allowing theme creators to define variables and values independent of the site content.
  2. The new theme variable configuration user interface allows site admins to modify theme variable values in the GUI and have the changes displayed immediately on site.
  3. Theme variables can be user defined and used with existing or custom themes and skins.

Database changes include a new column and index on the Theme Variables table.
